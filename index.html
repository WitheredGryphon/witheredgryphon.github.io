Quick Note
If/when an official translation is released, please support the authors or developers of the official translation.

Furthermore, this translation is not professionally done. There are likely to be inaccuracies, typos, grammar issues, and there may even be blatant mistranslations. This is a one-man project, and I'm working with a very limited knowledge of the language. By using this patch you're acknowledging that all of these errors and/or bugs may be found within the patch. If you come across something that needs to be corrected, please send me a message on Github.

Installation Instructions

This translation requires you to have a complete copy of the game. The translation is not a substitute for the game.
Please support the developers by purchasing their game here:
https://www.dlsite.com/ecchi-eng/work/=/product_id/RE237469.html

Instructions:
1. Download the translation patch directly or from a mirror.
2. Extract the .RGSS3A file to your Black Souls II installation directory.
3. When asked, choose yes to overwrite both files.
4. After the overwrite is successful, the game should be in English.

Note: Saves are unaffected by future patches to the translation.
Note 2: For installation, you must extract Black Souls II using a Japanese Code Page.
